## Run

To run this application, you have to set your own environmental variables. Below are the variables that you need to set in order to run the application:

BASE_URL

SANITY_PROJECT_ID

SANITY_DATASET

SANITY_TOKEN

EMAILJS_SERVICE_ID

EMAILJS_TEMPLATE_ID

EMAILJS_PUBLIC_KEY

Now you can run app in the terminal.

```bash
cd nakel
npm run dev
```
